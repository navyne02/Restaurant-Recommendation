# Restaurant Recommendation System

A recommendation system that suggests restaurants to users based on their specific preferences.

## Features Used for Filtering
- Cuisine
- Price Range
- Aggregate Rating
- City

## How It Works
1. User enters preferences (cuisine, price, etc.)
2. System filters matching restaurants
3. Restaurants are ranked based on rating and popularity
4. Top recommendations are displayed

## Tech Stack
- Python
- Pandas

## Conclusion
The system effectively recommends restaurants based on user preferences using a content-based filtering approach, which improves recommendation quality.
